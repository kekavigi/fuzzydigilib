Tentang situs ini

Saya sering kesulitan mencari pustaka di Digital Library ITB yang relevan, secara cepat. Situs ini saya buat untuk mempermudah proses tersebut,tentu disesuaikan dengan opini pribadi apa yang saya anggap "penting".

Data yang digunakan situs ini adalah cache dari situs Digital Library ITB. Dengan demikian, setiap perubahan pustaka yang terjadi disana -- baik penambahan, perbaikan, maupun penghapusan -- tidak langsung terlihat di situs proyek ini. Namun, ada beberapa data yang saya ubah maupun tidak diikutkan, karena saya anggap kurang relevan. Contohnya mengetahui sebuah pustaka berasal dari fakultas FMIPA ketika pustaka tersebut diterbitkan oleh FMIPA - Matematika, adanya teks TIDAK ADA ABSTRAK pada data abstrak ¯\_(ツ)_/¯, mapun berapa banyak File dimiliki pustaka, yang jumlahnya dapat berubah tergantung pada akses yang diberikan. Detail bagaimana saya mengolah data tersebut dapat dilihat di repo GitHub saya.
